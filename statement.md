# Problem Statement

In today’s academic environment, students often struggle to understand how their study habits directly impact their performance in examinations. One of the most common questions among students is how many hours they should study to achieve a desired score. However, without proper analysis or data-driven insights, it becomes difficult to establish a clear relationship between study time and academic results.

The objective of this project is to develop a simple and effective Machine Learning model that can predict a student’s marks based on the number of hours they study. By using historical data and applying Linear Regression, the project aims to identify patterns and trends between study hours and marks obtained.

This system takes the number of study hours as input from the user and provides a predicted score as output. Additionally, it visualizes the relationship between these two variables through a graphical representation, helping users better understand how changes in study time can influence their performance.

The project is designed to be beginner-friendly while demonstrating the practical application of Machine Learning concepts such as data handling, model training, prediction, and visualization. It can serve as a foundational step for building more advanced predictive systems using real-world datasets.

Overall, this project helps in promoting data-driven decision-making for students and provides a clear insight into how consistent study efforts can impact academic success.
