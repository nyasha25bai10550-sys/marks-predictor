Marks Predictor

A simple Machine Learning project that predicts student marks based on study hours.

Features
	•	Predicts marks using Linear Regression
	•	Takes user input for study hours
	•	Shows predicted result instantly
	•	Displays graph (Study Hours vs Marks)
	•	Beginner-friendly implementation

Tech Used
	•	Python
	•	Pandas
	•	Scikit-learn
	•	Matplotlib

Files
	•	main.py → main program
	•	data.csv → dataset

How to Run
	1.	Install dependencies:
pip install pandas scikit-learn matplotlib
	2.	Run the program:
python main.py
	3.	Enter study hours to get predicted marks

Output
	•	Predicted marks in terminal
	•	Graph showing relation between study hours and marks
